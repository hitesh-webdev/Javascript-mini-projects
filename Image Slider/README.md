# Image Slider

A responsive image slider built using **HTML, CSS, and JavaScript** with the help of **Swiper.js** library.

## Features
- Autoplay slider
- Infinite loop
- Next / Previous navigation buttons
- Pagination dots
- Responsive layout

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Swiper.js (CDN)

## What I Learned
- Integrating third-party libraries
- Swiper.js configuration
- DOM-based UI components
- Responsive image handling

This project is part of my JavaScript mini projects while learning frontend development.
