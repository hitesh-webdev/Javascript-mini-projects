
# Quote Generator

A simple Quote Generator project made using HTML, CSS and JavaScript.
It fetches random technology quotes from an API and displays them on the screen.

## Features
- Generate random quotes
- Shows quote author
- Tweet the quote with one click

## Tech Stack
- HTML
- CSS
- JavaScript
- Fetch API

## API
Quotable API  
https://api.quotable.io/random?tags=technology
