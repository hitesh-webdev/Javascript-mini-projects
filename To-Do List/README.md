# To-Do List App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks are saved using browser localStorage
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
- Users can add tasks using the input field
- Clicking on a task marks it as completed
- Clicking the delete icon removes the task
- Tasks remain saved even after refreshing the page using localStorage

This project helped me practice:
- DOM manipulation
- Event delegation
- localStorage for data persistence
- Basic UI and interaction logic
