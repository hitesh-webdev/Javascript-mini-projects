
# Drag and Drop App

A simple drag and drop application built using HTML, CSS, and JavaScript.

## Features
- Drag items from one container to another
- Move items back and forth between sections
- Clean and responsive UI
- Smooth drag-and-drop interaction

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
- List items are made draggable using the HTML `draggable` attribute
- JavaScript drag events are used:
  - dragstart
  - dragover
  - drop
- The selected item is appended to the target container on drop

## Learning
This project helped me understand:
- Basic drag and drop events in JavaScript
- DOM manipulation using `appendChild`
- Handling user interactions with event listeners
- Improving UI with CSS layouts and styling
